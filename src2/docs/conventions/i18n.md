# i18n Conventions

## File Structure

Each module has its own translation files:

```
<module>.i18n.en.json
<module>.i18n.fr.json
```

## Rules

- All user-visible strings must use translation keys
- Keys are namespaced by module: `"notifications.drawer.title"`
- All keys must be sorted alphabetically within their JSON file
- No fallback default values in code — strict key usage only
- Add both `en` and `fr` translations when adding new keys
