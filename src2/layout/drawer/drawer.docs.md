# layout/drawer

Side drawer UI component providing a flexible panel interface for settings, filters, and detail views. Includes actions, content area, and controls for closing and maximizing.

## Responsibilities

- Drawer open/close state management
- Drawer chrome (header actions, close button, maximize button)
- Content area rendering with configurable width
- Drawer configuration and default settings

## Key Files

- `drawer.components.tsx` — Drawer UI components (header, content, actions)
- `drawer.config.ts` — Drawer configuration (width, position, defaults)
- `drawer.hooks.tsx` — Hooks for drawer state access and manipulation
- `drawer.layout.tsx` — Main drawer layout component
