# features/classification

Display utilities for classification markings. Parses and formats classification strings for UI rendering. Note: actual classification enforcement is handled server-side — this module is for display purposes only.

## Responsibilities

- Parse classification strings into structured display data
- Format classification levels for banners and badges
- Provide human-readable labels for classification markings

## Key Files

- `classificationParser.ts` — Classification string parsing and formatting utilities
- `classificationParser.test.ts` — Unit tests for the parser
