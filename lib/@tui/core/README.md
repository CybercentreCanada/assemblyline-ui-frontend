# @tui/core

The core, mandatory, module of TemplateUI.

This module provides the base implementation of the TemplateUI common look and feel.

## Installation

First create a `.npmrc` under `$HOME`

```
@tui:registry=https://bagofholding.cse-cst.gc.ca/repository/npm-ap/
```

This will ensure @tui npm packages get pulled from BoH.

Then install @tui/core

```bash
pnpm install @tui/core
```
